# PlayReady QA Automation Framework

## Enterprise-Grade RAG Chatbot Testing with Azure AI Foundry + RAGAS

### What This Framework Does

1. **Ingests 42 PlayReady PDF documents** — extracts text, creates chunks with metadata
2. **Generates 965 intelligent test cases** — using Azure OpenAI LLM (not templates)
3. **Calls the real Foundry Agent** — the same agent used in production
4. **Evaluates quality** using RAGAS (13 metrics) + Azure AI Evaluation SDK (12+ metrics)
5. **Generates audit-ready Excel reports** with PASS/FAIL, color coding, and thresholds
6. **Validates 100% test coverage** across all PDFs and scenario types

## Architecture

```text

        │
        ▼

        │
        ▼

Quick Start (New Team Member)

Step 1: Clone and setup

git clone https://github.com/Sushrut051220/playready-qa-automation.git
cd playready-qa-automation
python -m venv .venv
.venv\Scripts\Activate.ps1
.venv\Scripts\pip.exe install -r requirements.txt

Step 2: Configure credentials

code .env

Set these values (get from team lead):
AZURE_OPENAI_API_KEY=
AZURE_OPENAI_BASE_URL=
AZURE_OPENAI_CHAT_DEPLOYMENT=
AZURE_OPENAI_API_VERSION=
AZURE_OPENAI_EMBEDDING_DEPLOYMENT=


Leave AZURE_OPENAI_API_KEY empty to use Azure Identity (az login).

Step 3: Azure login

az login

Step 4: Copy PlayReady PDFs
Copy-Item "\\shared\playready-pdfs\*.pdf" -Destination "data\kb\"
dir data\kb\*.pdf

Step 5: Generate everything and run

$env:PYTHONPATH="."

# PDF setup
.venv\Scripts\python.exe scripts/rename_pdfs.py
.venv\Scripts\python.exe scripts/regenerate_registry.py

# Generate chunks from PDFs
.venv\Scripts\python.exe scripts/generate_ragas_testset.py

# Generate smart test cases (LLM-powered, ~30-45 min)
.venv\Scripts\python.exe scripts/generate_smart_testcases.py

# Add negative + conversational tests
.venv\Scripts\python.exe scripts/generate_negative_testcases.py
.venv\Scripts\python.exe scripts/generate_conversational_testcases.py

# Validate 100% coverage
.venv\Scripts\python.exe scripts/validate_test_coverage.py

# Run pipeline
.venv\Scripts\python.exe scripts/query_foundry_agent.py --delay 2
.venv\Scripts\python.exe -m pytest -m ragas -v -s
.venv\Scripts\python.exe -m pytest -m foundry_eval -v -s

# Open reports
Start-Process reports\bridge\Test_Coverage_Report.xlsx
Start-Process reports\bridge\Bridge_Evaluation_Report.xlsx
Start-Process reports\bridge\Foundry_Evaluation_Report.xlsx
